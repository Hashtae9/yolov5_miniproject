## 결론 및 아쉬운점
+ 하이퍼파라미터 수치 조정을 통해 겹쳐진 이미지에 대한 인식이 더 향상되었음
+ 하지만 인식이 잘된 만큼 지폐를 다른 화폐로 인식하는 경우도 존재
  + 영상을 보면 5000원을 50000원으로 인식하는 등 가끔 다른 화폐로 오인
  + 하지만 꾸준하게 화폐의 위치를 잡는 모습을 확인 가능
